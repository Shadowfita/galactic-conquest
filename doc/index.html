Getting Started

Here is some basic documentation written by Orion to get you started with mods for Galactic Conquest. It will be expanded upon soontm

Using Mods

With the mod management system implemented, it is easy to create, load, and share mods. To add a mod to the game, drag the mod folder into the "mods" folder in the main game directory, then click on the mod in the game settings menu.

Mods will eventuallytm, in later updates, be able to change almost every aspect of the game, including gameplay, sound, and graphics.

Creating Mods

The creation of mods is for people who know their way around JavaScript, HTML, CSS, JSON, and basic game design fundamentals. This documentation contains the steps and available APIs that can be used. If you are just starting out, take a look at the Template mod.

Mod Template

The mod template serves as a starting point for creating a mod.

Content

Each mod contains content that replaces the default. This allows a mod to have control over how the game looks, sounds, and plays. The Template contents can be found in main game directory in mods/Template. These assets include images, music, and JSON data. The required files include:

  • info.json
  • music.json
  • planets.json
  • emoticons.json
  • img.jpg
  • icon.png

These files can be left untouched if you do not need to change them. Other assets can be added an loaded from either the main mod directory or other subdirectories.

Creating a New Mod

Duplicate the Template folder in the mods folder and rename it to what you want your mod to be named. You can now edit the assets inside freely and add new ones if needed. JSON files can be edited with any text-editing program, while images and sound files require programs like Photoshop or Audacity respectively.

Mod Structure

The different files in a mod must be structured according to this documentation. Deviation from the standard could result in a non-functional or game-breaking mod.

info.json

{
    "author": "Name of Mod Creator",
    "description": "Mod description goes here."
}

Right now, the info.json file only contains the description and author values. Multiple authors can be listed as long as the value is a single string. This may be added on to in the future.

music.json

[
    "Song 1",
    "Song 2",
    "Song 3",
    "Song 4"
]

The music.json file contains a list of the available songs to be played on the main menu. A random song from the array is selected when the mod is loaded and will change to a different song after one song is completed, automatically.
